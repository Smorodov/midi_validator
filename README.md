# MIDI Validator v7.0.0

Инструмент командной строки для проверки MIDI файлов на соответствие стандарту MIDI 1.0.

## Возможности

- Проверка структуры MIDI файлов (форматы 0, 1, 2)
- Валидация всех типов MIDI сообщений
- Проверка диапазонов значений (ноты, скорость, контроллеры)
- Поддержка мета-событий
- Обработка бегущего статуса (running status)
- Детальная диагностика ошибок
- Несколько форматов вывода

## Установка

### Требования
- Windows (MinGW).
- CMake 3.16+
- Компилятор с поддержкой C++17

### Сборка

```bash
git clone https://github.com/Smorodov/midi_validator.git
cd midi_validator
build.cmd
